What is iMovie? iMovie is one of the core program’s in Apple’s iLife suite. It is a linear movie editing program, which means that clips are edited on a timeline, showing the length of the project. We will be using iMovie today to turn some clips into our own movie project. After this session, you will know how to work with the four basic elements in an iMovie.
