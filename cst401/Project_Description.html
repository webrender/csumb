A Mobile Code Development Environment


Codile aims to be a mobile web development environment, for the scripter who is always moving. With Codile, web development and site management fits right into the developer's pocket. Developers can connect to their remote server, make changes to their site, preview the changes, and publish them -  all from their iPhone.


The Codile project will be a web application for the Apple iPhone,  which is split into four different sections, or Views.  These views will allow for a robust web development platform on the iPhone, while keeping the look and feel of the application clean and concise.  The four views are:


  1. •File View - an FTP client, to view files on a remote server


•Code View - an iPhone-formatted editor, with AJAX code updates


•Web View - a live web preview of the site you are editing, to see how the changes you have made to code look, before they are published live


•Reference Guide - a web languages reference, formatted for the iPhone, and containing information about common tags in HTML and CSS.
