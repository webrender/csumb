Self Assessment 5
 
Design Process:  I was really, really comfortable with this assignment.  I’ve been doing websites since i hit puberty, and so to get a first assignment which handled the actual design of a site was really fun for me.  I actually have a confession, I barely used any sketches before i started putting my work down in photoshop - all i really need to think about before i start a site is where the elements are going to go, i usually like to handle things like size and color of the site in my web designer, se that i can see how its going to look on an actual computer, rather than just down on paper.  
 
Degree of Difficulty: This assignment didn’t really take me out of my “comfort level” - i am so used to designing sites that i didn't have that much of a problem coming up for designs for the sites - much less than i had coming up with ideas for the other assignments we were given. I only had 4 or 5 different design ideas before i settled on the final three, it didn’t take me long to figure out which designs conveyed the information best.
 
Technique and Production:  I am very comfortable with the tools and options available in Photoshop,  and i know Dreamweaver like the back of my hand - i’ve been using it since Dreamweaver 2.  Translating the site from Photoshop to Dreamweaver wasn’t really a problem for me, since i had designed the site in the first place with the HTML code in mind - i already knew how it would be designed before it was completed in Photoshop.  
 
Reflection:  I had a lot of fun putting this module together, but i dont think i really learned many new concepts from it - oh well, experience is learning, right? :)
