Production Assignment 2
 
 
In this assignment you will use the colors you selected for Design Assignment 6 to create three prototype pages that apply your color system to the different text formats on a webpage.  You will need to select the color for the page background, colors for the various heading formats, and the color for the body/paragraph text.
