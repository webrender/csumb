Codile aims to be a mobile web development environment, for the developer who is always on the move.  The Codile application will run on the iPhone, and is split into four main views. The File View will manage an FTP connection and allow for quick file management. 
The Code View will provide a full-featured code editor, for quick edits on the iPhone. The Preview View will allow for quick viewing of files, or changes made in preview mode.  Finally, the Reference View will provide a full web language reference in an easy to use format which can be used complementary to the rest of the program, or used standalone as a quick reference guide.  Previously, a developer would need a laptop and several programs to manage all these functions;  with Codile, all of this power is available in one’s pocket.
 

Web development.  In your pocket.
